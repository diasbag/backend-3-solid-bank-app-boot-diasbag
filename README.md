# SolidBankApp
 ## SOLID Bank app запуск программы
 1. Склонировать себе репозитории `https://github.com/diasbag/BankApp`
 
 2. Открыть в среде разработки 

 3. Запустить проект
 
 ## Команды для работы в терминале 
 _1 - show accounts_
  
 _2 - create account_
 
 _3 - deposit_
                
 _4 - withdraw_
                
 _5 - transfer_
 
 _6 - this message_
 
 _7 - exit_
